File: db_config.php
```php

$db_user = 'root';  // Database username
```

### Database Connection Usage
The following files use the database connection constants to establish connections:

1. direct_image.php
```php
$db = new mysqli(DB_HOST, DB_USER, DB_PASS, DB_NAME);
```

2. image_proxy.php
```php
$db = new mysqli(DB_HOST, DB_USER, DB_PASS, DB_NAME);
```

3. lib/functions/common.php (doDBConnect function)
```php
$result = $db->connect(DSN, DB_HOST, DB_USER, DB_PASS, DB_NAME);
```

4. lib/api/xmlrpc/v1/xmlrpc.class.php (TestlinkXMLRPCServer._connectToDB function)
```php
$this->dbObj->connect(DSN, DB_HOST, DB_USER, DB_PASS, DB_NAME);
```

### Configuration Generation
install/installNewDB.php is responsible for generating the config_db.inc.php file during installation.
