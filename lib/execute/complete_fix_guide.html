❌ Issue 2: Tooltip Repeating Information

Cause: Complex tooltip logic showing all datasets instead of just hovered dataset

✅ Fix 2: Simplify Tooltip

Location: In the tooltip callbacks section, replace the entire label function.

Find this complex code (around lines 1817-1858):

label: function(context) { var label = context.dataset.label || ''; var value = context.parsed.y; var deltaIndex = context.dataIndex; // Get both values for calculation var executed = window.currentCumulativeExecuted[deltaIndex]; var passed = window.currentCumulativePassed[deltaIndex]; var assigned = window.currentAssignedData[deltaIndex]; // Calculate delta percentage correctly var deltaValue = executed - passed; var deltaPercentage = ((deltaValue) / executed * 100).toFixed(1); if (label.includes('Executed')) { return [ 'Executed: ' + executed, 'Passed: ' + passed, 'Assigned: ' + assigned, 'Delta: ' + deltaValue, 'Delta %: ' + deltaPercentage + '%' ]; } else if (label.includes('Passed')) { return [ 'Passed: ' + passed, 'Executed: ' + executed, 'Assigned: ' + assigned, 'Delta: ' + deltaValue, 'Delta %: ' + deltaPercentage + '%' ]; } else if (label.includes('Assigned')) { return [ 'Assigned: ' + assigned, 'Executed: ' + executed, 'Passed: ' + passed, 'Remaining: ' + (assigned - executed), 'Completion %: ' + ((executed / assigned * 100).toFixed(1)) + '%' ]; } return label + ': ' + value; }

Replace with this simple code:

label: function(context) { var label = context.dataset.label || ''; var value = context.parsed.y; return label + ': ' + value; }

🎯 Expected Result After Fixes

  • ✅ Numbers on each point - datalabels plugin shows values
  • ✅ Clean tooltips - only shows hovered dataset info
  • ✅ No repetition - single line per tooltip
  • ✅ Bold numbers - better visibility
