Redmine Bug Fix Bookmarklet for TestLink

This bookmarklet will scan your TestLink page for bug IDs (numbers starting with 64 followed by 3-5 digits) and convert them into clickable links to the Redmine issue tracker.

Drag this button to your bookmarks bar:

Fix Redmine Bug IDs

How to Use

  1. Drag the button above to your browser's bookmarks bar
  2. Navigate to any TestLink page that contains bug IDs
  3. Click the bookmarklet in your bookmarks bar
  4. Bug IDs will be converted to clickable links

Manual Method

If you can't use the bookmarklet, you can paste this code into your browser's console (F12 > Console):

(function(){var s=document.createElement('script');s.src='https://test-management.nmbtz.com/testlink/custom/js/redmine_simple_fix.js?v='+Math.random();document.body.appendChild(s);})();

Troubleshooting

  • If you don't see any changes, check if there are any bug IDs on the page (numbers starting with 64 followed by 3-5 digits)
  • Look for a small indicator in the bottom right corner of the page showing the status of the fix
  • Check your browser's console (F12 > Console) for any error messages
